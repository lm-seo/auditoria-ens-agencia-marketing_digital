# Auditoría ENS – Agencia de Marketing Digital
Este repositorio documenta un caso práctico completo de auditoría del Esquema Nacional de Seguridad (ENS) aplicado a una agencia de marketing digital ficticia (Sayonara), con el objetivo de demostrar competencias prácticas en GRC / Seguridad de la Información.

El trabajo simula una auditoría interna ENS, siguiendo criterios reales utilizados en consultoría y alineados con las guías CCN-STIC.

## 2. Objetivo del proyecto

- Evaluar el nivel de cumplimiento del ENS

- Identificar riesgos y brechas de seguridad

- Proponer un plan de adecuación realista

- Generar evidencias reutilizables como portfolio profesional GRC
